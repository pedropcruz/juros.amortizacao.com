# Juros - Simulador de Crédito Habitação

[![Nuxt UI](https://img.shields.io/badge/Made%20with-Nuxt%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.nuxt.com)

**Juros** é a ferramenta definitiva para quem quer entender e dominar o crédito habitação em Portugal. Desenvolvido com [Nuxt UI](https://ui.nuxt.com), oferece uma forma simples e intuitiva de simular empréstimos, comparar cenários e planear amortizações antecipadas.

## Visão do Projeto
Capacitar os utilizadores com literacia financeira, desmistificando o crédito habitação e permitindo prever, simular e planear os pagamentos de forma transparente. O objetivo é ajudar a poupar milhares de euros em juros.

## Funcionalidades
- Simulação detalhada de crédito habitação (Sistema Francês)
- Análise da composição das prestações (Juros vs. Capital)
- Simulador de amortizações antecipadas
- Comparação side-by-side de até 3 simulações
- Exportação PDF profissional
- Visualização interativa com gráficos

## Quick Start

```bash
# Clone this repository
git clone https://github.com/pdcrz/juros-amortizacao.git

# Navigate to the project directory
cd juros-amortizacao

# Install dependencies
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
bun dev
```

## Production

Build the application for production:

```bash
bun build
```

Locally preview production build:

```bash
bun preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
